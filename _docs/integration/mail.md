АРК: Транспортная логистика Online поддерживает отправку сообщений на почту от домена организации.

## Как подключить отправку?

1. Предварительно должна быть выполнена процедура проверки владения доменом. Для этого по запросу для вашего домена администратор «АРК: Транспортная логистика Online» пришлет значение ресурсной записи типа TXT postbox._domainkey.example.com для внесения в вашей доменной зоне у регистратора. После подтверждения домена пришлет имя и пароль

2. Перейдите в `Организация → Наши компании → Настройки → ЭОД`. В разделе `SMTP Yandex Cloud Postbox` внесите данные, полученные от администратора
